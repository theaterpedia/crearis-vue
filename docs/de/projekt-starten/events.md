# Schritt 1: Events

Im ersten Schritt fügst du **Veranstaltungen** zu deinem Projekt hinzu.

## Was sind Events?

Events sind die Kern-Einheiten deines Projekts:
- Workshops
- Kurse
- Aufführungen
- Projektwochen
- Seminare

## Events hinzufügen

<div class="screenshot-placeholder">
Screenshot: Event-Auswahl im Stepper
</div>

1. Klicke auf **"Add Event"**
2. Wähle eine Event-Vorlage aus der Liste
3. Passe die Details an

### Event-Vorlagen

Theaterpedia bietet vorbereitete Vorlagen für typische theaterpädagogische Formate:

| Vorlage | Beschreibung |
|---------|-------------|
| Workshop | Einzelner Workshop (2-4 Stunden) |
| Kurs | Mehrtägiges Format |
| Projektwoche | Schulprojekt über mehrere Tage |
| Aufführung | Öffentliche Präsentation |

## Event-Details bearbeiten

Nach der Auswahl kannst du bearbeiten:

- **Titel** - Name der Veranstaltung
- **Beschreibung** - Worum geht es?
- **Datum & Zeit** - Wann findet es statt?
- **Ort** - Wo findet es statt?
- **Kapazität** - Wie viele Teilnehmer?
- **Preis** - Kosten für Teilnehmer

</div>

## Events verwalten

Nach dem Hinzufügen siehst du deine Events in einer Liste:
- **Bearbeiten** - Details ändern
- **Duplizieren** - Vorlage für ähnliche Events
- **Löschen** - Event entfernen

::: warning Hinweis
Gelöschte Events können nicht wiederhergestellt werden.
:::
