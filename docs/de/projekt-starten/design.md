### Verfügbare Themes

Jedes Theme hat:
- Eigene Farbpalette
- Typografie-Einstellungen
- Layout-Varianten

::: tip oklch-Farbsystem
Theaterpedia verwendet das moderne **oklch**-Farbsystem für konsistente Farben auf allen Geräten.
:::

## Farben anpassen

### Primärfarbe
Die Hauptfarbe deines Projekts - erscheint bei:
- Buttons
- Links
- Akzenten

### Sekundärfarbe
Für unterstützende Elemente:
- Badges
- Highlights
- Hover-Effekte

## Typografie

### Schriftarten

Theaterpedia nutzt die **Monaspace**-Schriftfamilie:
- MonaspaceNeon (Standard)
- MonaspaceArgon
- MonaspaceKrypton
- MonaspaceRadon
- MonaspaceXenon

Alternative: **Roboto** für bessere Lesbarkeit bei langen Texten.

## Layout-Optionen

### Seitenbreite
- Schmal (900px)
- Standard (1200px)
- Breit (1440px)

### Border-Radius
- Keine Rundungen
- Klein (0.25rem)
- Mittel (0.5rem)
- Groß (1rem)

## Theme speichern

1. Klicke auf **"Theme speichern"**
2. Das Theme wird für dein Projekt aktiviert
3. Alle öffentlichen Seiten nutzen das neue Design

::: warning Beta-Feature <span class="beta-badge">v0.5</span>
Erweiterte Theme-Anpassungen werden in v0.5 verfügbar sein.
:::
